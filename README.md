# Planet Game

シンプルながら奥が深い、HTML5 Canvasを使用したウェブベースの回避アクションゲームです。惑星とその衛星を避け、ゴールを目指します。

## 🎮 プレイURL
[https://sin1.studio/Planet-Game/](https://sin1.studio/Planet-Game/)

## 🚀 ゲーム概要
自機を操作して、敵機（惑星）と、その周囲を公転する衛星を避けながら進みます。
ステージは全7ステージ。最終ステージには巨大なボスが待ち受けています。

### 操作方法
- **タップ / クリック**: 自機をその場所へ移動させます。

### 基本ルール
- **移動ペナルティ**: 画面をタップ/クリックするたびに **BONUSが10減少** します。
- **時間ペナルティ**: 時間が経過するにつれてBONUSが徐々に減少します。
- **ゲームオーバー**: 敵機・衛星に接触するか、**BONUSが0** になると失敗です。
- **スコア**: 各ステージクリア時の残りBONUSがSCOREに加算されます。

## 🪐 特徴
- **アダプティブ・レイアウト**: 
  - 横長画面（PCなど）: 左から右へ進む
  - 縦長画面（スマホなど）: 上から下へ進む
- **索敵AI**: 敵機はある程度自機に近づくまで追尾を開始しません。
- **分散配置システム**: 画面全体にまんべんなく敵機が配置されるアルゴリズムを搭載。
- **ボスバトル**: ステージ7では、ランダムに動き回る巨大なボスが出現します。ボス本体に触れればクリアですが、大量の衛星が防御壁となっています。
- **消滅エフェクト**: 自機の破壊、ゴールの爆発、ボスの撃破時などにパーティクル演出が発生します。
