# ORBITAL_DRIFT

タップ操作で自機を誘導し、エネルギーを管理しながら惑星の引力と衛星の軌道を回避するアクションゲームです。

## 🪐 特徴
- **エネルギーシステム**: 移動や時間経過でエネルギーを消費します。エネルギーが切れると操作不能になり、最も近い敵が高速であなたを仕留めにやってきます。
- **ダイナミック・チェイス**: エネルギー切れの際、遠くにいる敵は通常の3倍の速度で急接近し、緊張感のある最期を演出します。
- **ビジュアル演出**: 移動速度に応じた残像エフェクトや、自機の近くに表示されるリアルタイム・エネルギーインジケーター。
- **ボスバトル**: ステージ5では高速で不規則に漂う巨大なボスが登場。

## 🚀 ルール
- **目的**: 敵機や衛星を避け、ゴールへ到達。
- **移動**: タップ/クリック。エネルギーを10消費します。
- **スコア**: 各ステージクリア時の残りエネルギーがスコアとして加算。
